## ヒアリングスクリプト

### 導入（5分）

```
いま僕がやってる仕事の延長で、新しいサービスを考えていて、
率直な意見を聞かせてほしいんですけど、15分くらいお時間いいですか？

売り込みじゃないので、「いらない」って思ったら遠慮なく言ってください。
そのフィードバックが一番ありがたいです。
```

### 課題の深掘り（10分）

聞くこと（この順番で）:

1. **「月次レポートって今どうやって作ってますか？」**
   - 誰が / 何時間かけて / どのデータを / どこに出力してるか
   - 具体的な作業手順を聞く（抽象論ではなく）

2. **「それ、面倒だなって思うことありますか？」**
   - 不満の温度感を測る。「まあ仕方ない」なのか「本当に嫌」なのか
   - 金額換算: その作業、外注するとしたらいくら払いますか？

3. **「広告とか売上のデータ、日次で見てますか？」**
   - 見てる → どうやって？ツールは？
   - 見てない → なぜ？（時間がない / やり方がわからない / 見ても意味がない）

4. **「AI使って何か自動化したいなって思ったことあります？」**
   - ある → 具体的に何を？なぜまだやってない？
   - ない → AIに対するイメージは？

### 提案とフィードバック（5分）

```
実は僕、今やってるEC13店舗の運用で、月次レポートを完全自動化したんですよ。
Sheetsのデータを読んで、AIが分析して、スライドに起こして、Slackに届く。
人間の作業ゼロで。

これを他の会社にも作れるサービスを考えてるんですけど、
もしこういうの導入するとしたら:

- いくらなら検討しますか？（具体的な金額を聞く）
- 最初に「業務診断」だけ10万円でやるとしたら、試してみたいですか？
- 逆に、「それより先にこっちを自動化してほしい」ってのあります？
```

### クロージング（2分）

```
めちゃくちゃ参考になりました。ありがとうございます。
もし興味あれば、もう少し具体的な提案書持ってきますけど、見てみたいですか？
```

**聞いた後に必ず記録すること:**
- [ ] 3つの予想外（想定と違ったこと）
- [ ] 修正すべき仮説
- [ ] 次に会うべき人（紹介してもらえそうな人）
- [ ] 価格に対するリアクション（具体的な言葉をそのまま記録）

## 検証スケジュール

| 日程 | やること |
|------|---------|
| Day 1-3 | 最小コストテスト（X投稿 + note記事 + BPG紹介依頼） |
| Day 4-7 | ヒアリング日程調整 + 最初の1-2人実施 |
| Day 8-14 | ヒアリング3-5人追加実施 |
| Day 14 | **GO / PIVOT / KILL 判定** |

### GO基準
- 5人中2人以上が「金を払う」と明言（検討する、ではなく）
- または1人でも「今すぐ発注したい」と言った場合

### PIVOT基準
- 「レポート自動化」には金を払わないが、別の業務に強い需要がある場合
- 価格帯が大幅にずれている場合（→ 価格調整してリトライ）

### KILL基準
- 5人全員が「面白いけど今は要らない」
- 具体的な業務課題が出てこない（そもそも困ってない）
