# サービス設計 v1（Day 2 / Phase 1-2）

## サービス名（仮）

**AI業務自動化パートナー**
（一行: 中小企業の属人業務をAIエージェントで自動化する）

## 提供メニュー（3段階）

### Menu A: 業務診断 + 設計（入口商品）

| 項目 | 内容 |
|------|------|
| 提供内容 | 業務フロー可視化 → 自動化対象の特定 → AIエージェント設計書 |
| 工数 | 2-3日（ヒアリング1日 + 設計2日） |
| 価格 | **10-15万円** |
| 目的 | 低リスクで試してもらう入口。ここで信頼を獲得し、Menu Bに繋げる |
| 納品物 | 業務フロー図（Mermaid）、自動化ROI試算、AIエージェント設計書 |

### Menu B: AIエージェント構築（主力商品）

| 項目 | 内容 |
|------|------|
| 提供内容 | 設計書に基づくAIエージェント構築 → テスト → 本番投入 |
| 工数 | 1-2週間（案件複雑度による） |
| 価格 | **30-80万円** |
| 目的 | メインの収益源。1件の利益 = 25-70万円（原価はAPI費のみ） |
| 納品物 | 稼働するAIエージェント、運用マニュアル、30日間の無料サポート |

#### Menu B のバリエーション

| パターン | 価格帯 | 具体例 | 工数 |
|----------|--------|--------|------|
| **B-1: レポート自動化** | 30-50万円 | Sheets→Claude→Slides→Slack | 1週間 |
| **B-2: データダッシュボード** | 30-50万円 | 日次収集+ダッシュボード+BigQuery | 1週間 |
| **B-3: マルチエージェント** | 50-80万円 | 5-10体のエージェント+スキルセット | 1-2週間 |
| **B-4: 競合リサーチ自動化** | 40-60万円 | 競合データ収集→分析→レポート | 1週間 |

### Menu C: 月額運用保守（ストック収入）

| 項目 | 内容 |
|------|------|
| 提供内容 | 月次改善、障害対応、追加エージェント開発、Slack相談 |
| 価格 | **月額5-15万円** |
| 目的 | MRR（月次定期収入）の積み上げ。3社で月15-45万円のベース収入 |
| 含まれるもの | 月2回の定例MTG、月1回の改善提案、軽微な追加開発（月5h以内） |

## 市場相場との比較

| 依頼先 | 初期費用 | 月額 |
|--------|----------|------|
| 大手ベンダー | 数十〜数百万円 | 5-150万円 |
| フリーランス委託（相場） | 50-500万円 | 5-50万円 |
| **自社（Menu B）** | **30-80万円** | **5-15万円** |
| SaaSプラットフォーム | 0-50万円 | 1-30万円 |

**ポジショニング: フリーランス相場の最安圏 × 実績で信頼を取る**

自社の強み:
- 大手ベンダーの1/10以下の初期費用
- SaaSと違い、完全カスタマイズ可能
- 13店舗・20エージェント・43スキルの本番稼働実績
